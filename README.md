# Building AI – University of Helsinki (Elements of AI)

A collection of algorithm implementations and exercises completed as part of the
**[Building AI](https://buildingai.elementsofai.com/)** course by Reaktor and the
University of Helsinki.

> *Building AI is a free online course where you'll learn about the actual algorithms
> that make creating AI methods possible.*

---

## 📚 Course Structure

### Chapter 1 – Getting Started with AI
- Why AI matters
- Optimization
- Hill climbing

### Chapter 2 – Dealing with Uncertainty
- Probability fundamentals
- The Bayes rule
- Naïve Bayes classifier

### Chapter 3 – Machine Learning
- Linear regression
- The nearest neighbor method
- Working with text
- Overfitting

### Chapter 4 – Neural Networks
- Logistic regression
- From logistic regression to neural networks
- Deep learning

### Chapter 5 – Conclusion
- Summary & beyond

---

## 🛠️ Technologies Used

- **Python 3**

## 🚀 Getting Started

Clone the repository and navigate to the chapter of your choice:

```bash
git clone https://github.com/Akintola/building-ai-helsinki-university.git
cd building-ai-helsinki-university
```

Run any exercise file directly with Python:

```bash
python chapter1/hill_climbing.py
```
